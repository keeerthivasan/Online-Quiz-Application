🧠 Online Quiz Application

📌 Overview

This is a simple and interactive Online Quiz Application developed using HTML, CSS, and JavaScript. The application allows users to answer multiple-choice questions and displays the final score at the end. It is designed with a clean user interface and responsive layout.


🎯 Objective

The main objective of this project is to provide an easy way to conduct quizzes online and evaluate user performance instantly.


🚀 Features

- Multiple-choice questions
- Timer functionality
- Question navigation
- Score calculation
- Result display
- Responsive design
- Clean and simple UI


🛠️ Technologies Used

- HTML – Structure of the application
- CSS – Styling and layout
- JavaScript – Logic and functionality

⚙️ How It Works

Users start the quiz and answer each question by selecting an option. After completing all questions, the system calculates the score and displays the result. A timer is included to limit the quiz duration.

📂 Project Structure

- index.html – Contains HTML, CSS, and JavaScript code


🎓 Learning Outcomes

- Understanding of DOM manipulation
- Form handling and event handling
- Timer implementation in JavaScript
- Building responsive web applications
